# FICOFORCE Data Card

## Task 1 Data
`HACKATHON_TRAINING_DATA.CSV` contains account, loan, transaction, credit bureau, product, and target fields for ML-based fraud/default-risk detection.

The file is intentionally ignored by Git because it is large and may contain sensitive or restricted competition data.

## Task 2 Data
`Dummy Dataset Final.txt` contains synthetic borrower identity and location signals for local RAG-based verification.

## Sensitive Fields
Phone numbers, addresses, email addresses, driving license numbers, vehicle numbers, and social URLs should be masked before public demos.

## Recommended Public Demo Practice
Use synthetic rows or redacted screenshots in the README. Do not publish raw customer-level records.
