# FICOFORCE Frontend

React dashboard for the FICOFORCE fraud review workflow.

## Screens

- Account review hero form
- Task 1 ML fraud-risk scoring panel
- Task 2 RAG location verification panel
- Model metric cards from the backend
- Agent evidence cards for Task 2

## Run

```bash
npm install
npm start
```

Set the backend URL with:

```bash
REACT_APP_API_BASE=http://localhost:8000
```
